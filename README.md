# MCP CV Generator 🚀

**AI-powered CV/Resume Generator using Model Context Protocol (MCP) with Claude Desktop integration.**  
Supports file or text input and generates structured CV sections dynamically using modular tools.

---

## 🔥 Features

- 📂 **Input Support:** Analyze existing CV files or raw text prompts  
- 🧠 **Dynamic Content Generation:** Modular MCP tools generate each CV section  
- 🧩 **Tools Included:**
  - Summarization (Professional Summary)
  - Education Extraction
  - Experience Generation
  - Projects Generation
  - Skills Extraction
  - Certifications & Awards Extraction
  - Profile Builder (combines all sections)
- ⚡ **Output:** ATS-friendly, professional CV profile
- 🔗 Fully compatible with **Claude Desktop** MCP tools

---

## 🛠️ Tech Stack

- Python  
- FastMCP  
- Claude Desktop (MCP integration)  
- Prompt Engineering  

---

## ⚙️ How It Works

1. Input CV data (file or text)  
2. Each MCP tool generates a section:  
   - **Summary** → concise paragraph  
   - **Education** → degree, university, dates  
   - **Experience** → roles, achievements, bullets  
   - **Projects** → key projects and outcomes  
   - **Skills** → categorized with proficiency  
   - **Certifications** → badges, awards, memberships  
3. **Profile Generator** combines all sections into a complete CV  
4. Claude Desktop executes the prompts to generate dynamic outputs
